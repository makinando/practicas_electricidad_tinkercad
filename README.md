## 🛠️ Herramientas utilizadas

- [TinkerCad Circuits](https://www.tinkercad.com) — simulador de circuitos eléctricos online
- [Moodle Centros](https://educacion.juntadeandalucia.es) — plataforma de entrega de tareas
- Google Drive con cuenta corporativa `@g.educaand.es` — documentación del alumnado

---

## 📋 Instrucciones para el alumnado

1. Entra en **TinkerCad** con tu código de clase y usuario.
2. Monta el circuito de cada práctica y comprueba su funcionamiento.
3. Abre un documento en **Google Drive** con tu cuenta `@g.educaand.es` llamado *Prácticas de electricidad*.
4. Escribe el título de cada práctica y pega una captura de pantalla del circuito montado.
5. Al terminar todas las prácticas, envía el documento a la tarea de **Moodle Centros**.
